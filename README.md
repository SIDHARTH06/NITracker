# NITracker

link to backend (straPy) :https://github.com/gautham227/Code_init-2023.git 

link to frontend: https://github.com/AmalkrishnaAS/chakra-ui-starter.git

```
pip install requirements.txt
```

do commands
```
yarn install 
yarn develop
```
on strapi server

yarn install

on frontend

python app.py
